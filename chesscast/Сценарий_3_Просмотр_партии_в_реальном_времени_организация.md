# Сценарий 3: Просмотр партии в реальном времени в рамках организации

| №   | Действие специалиста                                                     | Результат                                                                                      | Время, сек. |
| --- | ------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------- | ----------- |
| 1   | Открывает приложение в браузере                                          | Отображается главная страница приложения                                                       | 2           |
| 2   | Нажимает кнопку "Войти"                                                  | Открывается страница входа                                                                     | 1           |
| 3   | Вводит email и пароль участника организации, нажимает "Войти"            | Происходит аутентификация, пользователь перенаправляется в личный кабинет                      | 3           |
| 4   | Переходит в раздел "Организации"                                         | Отображается список организаций, где пользователь является участником                          | 2           |
| 5   | Выбирает нужную организацию, нажимает на неё                             | Открывается страница организации с информацией и списком активных игр                          | 2           |
| 6   | Просматривает список активных игр организации                            | Отображается список игр, которые сейчас проводятся в реальном времени                          | 3           |
| 7   | Выбирает интересующую игру из списка активных                            | Открывается страница просмотра игры с информацией об игре                                      | 2           |
| 8   | Система проверяет права доступа пользователя                             | Пользователь имеет доступ к просмотру игр организации, доступ разрешен                         | 1           |
| 9   | Система устанавливает WebSocket соединение для просмотра                 | Устанавливается соединение с сервером для получения потока данных                              | 2           |
| 10  | Система подключается к WebRTC комнате игры                               | Пользователь присоединяется к комнате трансляции игры                                          | 3           |
| 11  | Получает видеопоток в реальном времени                                   | Начинается передача видеопотока с камеры организатора игры                                     | 2           |
| 12  | Отображается интерфейс просмотра                                         | Показывается видео с доски, виртуальная доска с текущей позицией, список ходов, анализ позиции | 1           |
| 13  | Наблюдает за игрой в реальном времени                                    | Видеопоток обновляется, виртуальная доска синхронизируется с реальной доской                   | -           |
| 14  | Система детектирует новый ход на доске                                   | Ход отображается на виртуальной доске, добавляется в список ходов, запускается анализ позиции  | 2           |
| 15  | Получает обновленную информацию о позиции                                | Отображается оценка позиции, глубина анализа, рекомендуемый ход, обновляется виртуальная доска | 1           |
| 16  | Продолжает наблюдать за игрой                                            | Процесс повторяется для каждого нового хода                                                    | -           |
| 17  | Использует элементы управления (пауза, перемотка, если доступно)         | При необходимости может приостановить просмотр или перемотать к определенному моменту          | -           |
| 18  | Завершает просмотр, закрывает страницу или нажимает "Выйти из просмотра" | Закрывается WebSocket соединение, отключается от WebRTC комнаты, освобождаются ресурсы         | 2           |
| 19  | Возвращается на страницу организации                                     | Отображается страница организации с обновленным списком игр                                    | 1           |

**Общее время активных действий: ~25 секунд (без учета времени просмотра)**
