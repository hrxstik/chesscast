# Сценарий 4: Просмотр партии из истории в рамках организации

| №   | Действие специалиста                                          | Результат                                                                                    | Время, сек. |
| --- | ------------------------------------------------------------- | -------------------------------------------------------------------------------------------- | ----------- |
| 1   | Открывает приложение в браузере                               | Отображается главная страница приложения                                                     | 2           |
| 2   | Нажимает кнопку "Войти"                                       | Открывается страница входа                                                                   | 1           |
| 3   | Вводит email и пароль участника организации, нажимает "Войти" | Происходит аутентификация, пользователь перенаправляется в личный кабинет                    | 3           |
| 4   | Переходит в раздел "Организации"                              | Отображается список организаций, где пользователь является участником                        | 2           |
| 5   | Выбирает нужную организацию, нажимает на неё                  | Открывается страница организации с информацией и управлением                                 | 2           |
| 6   | Переходит в раздел "Логи организации" или "История игр"       | Отображается список завершенных партий организации с датами, участниками, результатами       | 2           |
| 7   | Просматривает список завершенных партий                       | Отображается таблица или список партий с фильтрами (дата, участники, результат)              | 5           |
| 8   | Применяет фильтры для поиска нужной партии (опционально)      | Список партий фильтруется по выбранным критериям                                             | 3           |
| 9   | Выбирает интересующую партию из списка                        | Открывается страница просмотра партии с детальной информацией                                | 2           |
| 10  | Система проверяет права доступа пользователя                  | Пользователь имеет доступ к истории игр организации, доступ разрешен                         | 1           |
| 11  | Система загружает данные партии с сервера                     | Загружаются сохраненные ходы, метаданные партии, информация об участниках                    | 2           |
| 12  | Отображается интерфейс просмотра партии                       | Показывается виртуальная доска, список всех ходов, информация о партии, элементы управления  | 1           |
| 13  | Просматривает начальную позицию                               | Виртуальная доска отображает начальную расстановку фигур                                     | 1           |
| 14  | Нажимает кнопку "Воспроизвести" или начинает просмотр ходов   | Запускается автоматическое воспроизведение ходов партии                                      | 1           |
| 15  | Система воспроизводит ходы последовательно                    | Ходы применяются к виртуальной доске с заданной скоростью, отображается текущий ход в списке | -           |
| 16  | Использует элементы управления (пауза, перемотка, скорость)   | Может приостановить воспроизведение, перемотать к определенному ходу, изменить скорость      | -           |
| 17  | Переходит к конкретному ходу в списке                         | Виртуальная доска обновляется, показывая позицию после выбранного хода                       | 1           |
| 18  | Запускает анализ позиции для выбранного хода                  | Система запускает Stockfish для анализа позиции, отображает оценку и рекомендуемый ход       | 3           |
| 19  | Получает результат анализа позиции                            | Отображается оценка позиции, глубина анализа, рекомендуемый ход, лучшая линия                | 1           |
| 20  | Просматривает комментарии к ходам (если есть)                 | Отображаются комментарии, сделанные во время игры                                            | 2           |
| 21  | Экспортирует партию в PGN формат (опционально)                | Скачивается файл с записью партии в стандартном формате                                      | 2           |
| 22  | Завершает просмотр, возвращается к списку партий              | Закрывается страница просмотра, возврат к списку истории игр                                 | 1           |
| 23  | Продолжает просмотр других партий или выходит                 | Может выбрать другую партию для просмотра или вернуться на главную страницу организации      | -           |

**Общее время активных действий: ~33 секунды (без учета времени просмотра партии)**
